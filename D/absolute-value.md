**Definition:** The **absolute value** of a real number $x \in \mathbb{R}$ is defined as:

$$\lvert x \rvert = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$

The absolute value measures the distance from $x$ to the origin on the real line. More generally, $\lvert x - y \rvert$ measures the distance between $x$ and $y$.

**Properties:** For all $x, y \in \mathbb{R}$:

1. $\lvert x \rvert \geq 0$, with $\lvert x \rvert = 0$ if and only if $x = 0$
2. $\lvert xy \rvert = \lvert x \rvert \lvert y \rvert$
3. $\lvert x + y \rvert \leq \lvert x \rvert + \lvert y \rvert$ (the **Triangle Inequality**)
