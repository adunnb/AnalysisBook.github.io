**Definition:** The **absolute value** of a real number $x \in \mathbb{R}$ is defined as:

$$|x| = \begin{cases} x & \text{if } x \geq 0 \\ -x & \text{if } x < 0 \end{cases}$$

The absolute value measures the distance from $x$ to the origin on the real line. More generally, $|x - y|$ measures the distance between $x$ and $y$.

**Properties:** For all $x, y \in \mathbb{R}$:

1. $|x| \geq 0$, with $|x| = 0$ if and only if $x = 0$
2. $|xy| = |x||y|$
3. $|x + y| \leq |x| + |y|$ (the **triangle inequality**)
