**Definition:** Let $f : A \to \mathbb{R}$. A point $c \in A$ is a **local maximum** of $f$ if there exists $\delta > 0$ such that:

$$f(c) \geq f(x) \quad \text{for all } x \in A \text{ with } \lvert x - c \rvert < \delta$$

Similarly, $c$ is a **local minimum** of $f$ if there exists $\delta > 0$ such that:

$$f(c) \leq f(x) \quad \text{for all } x \in A \text{ with } \lvert x - c \rvert < \delta$$

A point is a **local extremum** if it is either a local maximum or a local minimum. A local extremum $c$ is called an **interior extremum** if $c$ is in the interior of $A$, i.e. $c$ is not an endpoint of $A$.
