**Definition:** Let $A \subseteq \mathbb{R}$ be a non-empty set. A number $b \in \mathbb{R}$ is called an **upper bound** of $A$ if:

$$b \geq a \quad \text{for all } a \in A$$

The set $A$ is **bounded above** if an upper bound exists. A number $s \in \mathbb{R}$ is called the **supremum** (or least upper bound) of $A$, written $s = \sup A$, if:

1. $s$ is an upper bound of $A$, and
2. if $b$ is any upper bound of $A$, then $s \leq b$.
