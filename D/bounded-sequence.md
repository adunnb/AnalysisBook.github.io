**Definition:** A sequence $(a_n)$ of real numbers is **bounded above** if there exists $M \in \mathbb{R}$ such that $a_n \leq M$ for all $n \in \mathbb{N}$. It is **bounded below** if there exists $m \in \mathbb{R}$ such that $a_n \geq m$ for all $n \in \mathbb{N}$.

A sequence is **bounded** if it is both bounded above and bounded below, i.e. if there exists $M > 0$ such that:

$$\lvert a_n \rvert \leq M \quad \text{for all } n \in \mathbb{N}$$
